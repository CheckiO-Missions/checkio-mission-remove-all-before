Ok, what should I start from?

Find the index of wanted element in array with array.indexOf() method and assign it to the new variable. 

let ind = values.indexOf(b);

In case, the element is not in array, the method returns -1.

Ok, how to make a correct slice of the array?

For this purpose use array.slice() method with start from index ind you've found. 

values.slice(ind);
What to do in case if element is not in array?

As I previously said, in this case ind equals -1. And here is a little trick here. Let's look at possible combinations of values: variable ind and expression ind == -1.

If ind >= 0 (element is present in array) ind == -1 is false (or 0). Otherwise, if ind = -1, ind == -1 is true (or 1). When element is present in array, you make a slice from it, when absent - return the whole array, or you may say - make a slice from the start of the array, from 0 index!

Such behavior may be reached, if you add these two values as an argument of slice() method. To convert value into number use + operator right before expression. 

values.slice(ind + +(ind==-1));

In case element is in array you get such expression (index is 5 for example): 5 + (ind==-1 -> false -> 0) = 5, in other case (element is not in array, so ind equals -1): -1 + (ind==-1 -> true -> 1) = 0. That's exactly what you need!

So, how my final solution should look like?

You just need to wrap your code into function expression and return your slice.

Here is the whole solution:

function removeAllBefore(values: number[], b: number): number[] {

    let ind = values.indexOf(b);
    
    return values.slice(ind + +(ind==-1));
}

This solution uses ternary operator and anonymous function. In case array includes the element, function returns slice, otherwise - the whole array. 

var removeAllBefore = (values: number[], b: number): number[] =>
values.includes(b) ? values.slice(values.indexOf(b)) : values;

If element is in the array, the maximum is more or equal 0, otherwise slice starts from 0, since 0 > -1. 

function removeAllBefore(values: number[], b: number): number[] {

    return values.slice(Math.max(0, values.indexOf(b)));
}
