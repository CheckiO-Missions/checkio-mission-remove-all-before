Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

chodzi o znalezienie indeksu żądanego elementu w tablicy i zwrócenie wycinka tablicy zaczynającego się od indeksu.

Ok, od czego powinienem zacząć?

Proszę znaleźć indeks poszukiwanego elementu w tablicy za pomocą metody array.indexOf() i przypisać go do nowej zmiennej. 

let ind = values.indexOf(b);

W przypadku, gdy element nie znajduje się w tablicy, metoda zwraca -1.

Ok, jak utworzyć prawidłowy wycinek tablicy?

W tym celu proszę użyć metody array.slice() z początkiem od indeksu ind, który Pan znalazł. 

values.slice(ind);

Co zrobić w przypadku, gdy element nie znajduje się w tablicy?

Jak już wspomniałem, w tym przypadku ind równa się -1. I tutaj jest mała sztuczka. Proszę spojrzeć na możliwe kombinacje wartości: zmiennej ind i wyrażenia ind == -1.

Jeśli ind>= 0 (element jest obecny w tablicy) ind == -1 jest false (or 0). W przeciwnym razie, jeśli ind = -1, ind == -1 jest true (or 1). Jeśli element jest obecny w tablicy, proszę utworzyć z niego wycinek, jeśli go nie ma - proszę zwrócić całą tablicę, lub można powiedzieć - proszę utworzyć wycinek od początku tablicy, od indeksu 0!

Takie zachowanie można osiągnąć, jeśli doda się te dwie wartości jako argument metody slice(). Aby przekonwertować wartość na liczbę, należy użyć operatora + tuż przed wyrażeniem. 

values.slice(ind + +(ind==-1));

W przypadku, gdy element znajduje się w tablicy, otrzymamy takie wyrażenie (na przykład indeks wynosi 5): 5 + (ind==-1 -> false -> 0) = 5, w innym przypadku (element nie znajduje się w tablicy, więc ind równa się -1): -1 + (ind==-1 -> true -> 1) = 0. To jest dokładnie to, czego Pan potrzebuje!

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcji i zwrócić wycinek.

Oto całe rozwiązanie:

function removeAllBefore(values: number[], b: number): number[] {

    let ind = values.indexOf(b);
    
    return values.slice(ind + +(ind==-1));
}

Jasne, proszę pobrać!

To rozwiązanie wykorzystuje operator trójargumentowy i anonimową funkcję. W przypadku, gdy tablica zawiera element, funkcja zwraca wycinek, w przeciwnym razie - całą tablicę. 

var removeAllBefore = (values: number[], b: number): number[] =>
values.includes(b)? values.slice(values.indexOf(b)): values;

Jeśli element znajduje się w tablicy, maksimum jest większe lub równe 0, w przeciwnym razie wycinek zaczyna się od 0, ponieważ 0 > -1. 

function removeAllBefore(values: number[], b: number): number[] {

    return values.slice(Math.max(0, values.indexOf(b)));
}
