Не знаю навіть з чого почати...
Ну перш за все, тобі потрібно знайти на якій позиції в списку знаходиться "border".
Мені все ще потрібна деяка допомога...
Якщо ти відкриєш свою консоль python та напишеш: 
a = [1, 2, 3]
a.index(2)

Ти повинен отримати у відповідь "1", тому що елемент "2" знаходиться на позиції "1".

Як я можу це використати у своєму коді?
Тобі потрібно зробити зріз списку, використовуючи "list.index(border)" як перший елемент. А також не забуть про перевірку на те, чи входить "border" до списку.
Останнє запитання. Як має виглядати мій фінальний код? 
def remove_all_before(items: list, border: int) -> Iterable:
    if border in items:
        return items[items.index(border):]
    else:
        return items

Або ти можеш оформити його в один рядок:

def remove_all_before(items: list, border: int) -> Iterable:
    return items[items.index(border):] if border in items else items
