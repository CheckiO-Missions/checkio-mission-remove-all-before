Je n'ai aucune idée de la manière de commencer à résoudre cette mission.

Tout d'abord, vous devez trouver la position de border dans la liste. Essayez d'utiliser la méthode list.index().

J'ai besoin d'aide...

Donc, si vous ouvrez votre console python et que vous dites: 

a = [1, 2, 3]
a.index(2)

Vous devriez obtenir 1, car l'élément 2 est en position "1".

Comment puis-je utiliser cela dans mon code?

Vous devez prendre une tranche de la liste en utilisant list.index(border) comme premier élément. Et n'oubliez pas de vérifier si border se trouve dans la liste.

Une dernière question. À quoi ressemblera ma solution finale? 

def remove_all_before(items: list, border: int) -> Iterable:

    if border in items:
        return items[items.index(border):]
    else:
        return items

Ou vous pouvez le faire en une ligne

def remove_all_before(items: list, border: int) -> Iterable:

    return items[items.index(border):] if border in items else items
