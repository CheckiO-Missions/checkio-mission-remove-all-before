用array.indexOf() 方法在数组中找到想要的元素的索引，并将其赋值给新变量。 

let ind = values.indexOf(b);

如果元素不在数组中，方法返回-1 。

好的，如何对数组进行正确的切片？

为此使用array.slice() 方法，从您找到的索引ind 开始。 

values.slice(ind);

如果元素不在数组中该怎么办？

正如我之前所说，在这种情况下，ind 等于-1 。这里有一个小技巧。让我们看看可能的值组合：变量ind 和表达式ind == -1 。

如果ind>= 0 (元素存在于数组中)ind == -1 就是false (or 0).否则，如果ind = -1,ind == -1 就是true (or 1).当元素存在于数组中时，从元素中切片，当元素不存在时，返回整个数组，或者可以说从数组的0索引开始切片！

如果将这两个值作为slice() 方法的参数，就可以得到这样的结果。要将数值转换为数字，请在表达式前使用+ 操作符。 

values.slice(ind + +(ind==-1));

如果元素在数组中，您将得到这样的表达式（例如索引为5）：5 + (ind==-1 -> false -> 0) = 5, 在其他情况下（元素不在数组中，所以ind 等于-1 ）：-1 + (ind==-1 -> true -> 1) = 0.这正是您所需要的！

那么，我的最终解决方案应该是怎样的呢？

您只需将您的代码封装到函数表达式中并返回您的切片。

下面是整个解决方案：

function removeAllBefore(values: number[], b: number): number[] {

    let ind = values.indexOf(b);
    
    return values.slice(ind + +(ind==-1));
}

这个解决方案使用了三元运算符和匿名函数。如果数组包含元素，函数返回切片，否则返回整个数组。 

var removeAllBefore = (values: number[], b: number): number[] =>
values.includes(b)? values.slice(values.indexOf(b)): values;

如果元素在数组中，最大值大于等于0，否则切片从0开始，因为0 >-1。 

function removeAllBefore(values: number[], b: number): number[] {

    return values.slice(Math.max(0, values.indexOf(b)));
}
